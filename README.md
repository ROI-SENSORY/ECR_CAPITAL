# ECR-_CAPITAL
Herramienta de inteligencia de mercado de la metodogía ECR$ Capital, que convierte estrategias experienciales en activos financieros medibles. A través de la capitalización del comportamiento del cliente, para detectar y cerrar fugas de capital. A través del sistema ROI‑Sensory, se evalúa cómo cada microexperiencia impacta en la rentabilidad.
